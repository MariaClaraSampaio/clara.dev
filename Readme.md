## 💗 Sobre o projeto

O **Clara.dev** é meu portfólio pessoal criado para mostrar minha trajetória no desenvolvimento web, reunindo projetos, tecnologias e experiências adquiridas durante meus estudos.

O objetivo é construir uma interface moderna, responsiva e com uma identidade visual própria, unindo desenvolvimento Front-end e design.

# 💻 Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/MariaClaraSampaio/clara.dev.git
```

Entre na pasta:

```bash
cd clara.dev
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
